# ML_WorkSpace — Academic Machine Learning Projects

A collection of machine learning algorithms and preprocessing techniques implemented for academic coursework. Includes 10+ classification algorithms applied to real-world datasets.

## 📁 Repository Structure

ML_WorkSpace/
|
├── algorithms/          # ML algorithm scripts (.ipynb)
|
├── datasets/            # Dataset files (.csv , .xlsx)
|
├── .gitignore
|
└── README.md


## 🛠️ Tech Stack

| Category | Libraries |
|----------|-----------|
| Data Manipulation | `pandas`, `numpy` |
| Scientific Computing | `scipy` |
| Visualization | `matplotlib` |
| Machine Learning | `scikit-learn` |

## 🧠 Algorithms & Datasets

| Algorithm | Type | Dataset(s) Used |
|-----------|------|-----------------|
| K-Nearest Neighbors (KNN) | Instance-based | Iris, Wine |
| Decision Tree | Tree-based | Iris, Wine |
| Naïve Bayes | Probabilistic | Titanic |
| Logistic Regression | Linear | Breast Cancer, Diabetes, Student Academic Performance |
| Support Vector Machine (SVM) | Kernel-based | Bank, Glass, Seed |
| Multi-Layer Perceptron (MLP) | Neural Network | Letter Recognition, MNIST |
| Random Forest | Ensemble (Bagging) | Adult Income, Car Evaluation, Mushroom |
| AdaBoost | Ensemble (Boosting) | Customer Churn, Heart Disease |
| Bayesian Belief Network | Probabilistic Graphical | — |
| Hidden Markov Model | Sequential | — |

## ⚙️ Preprocessing

The `preprocessing` scripts handle:
- Missing value imputation
- Feature scaling (Standardization / Normalization)
- Categorical encoding (One-Hot / Label Encoding)
- Train-test splitting
